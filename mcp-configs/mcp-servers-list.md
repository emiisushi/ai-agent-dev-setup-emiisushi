# MCP Servers List

## 1. Rolldice
- Purpose: Game/interactivity MCP server for procedural decision-making (randomized dice rolls in agent workflows).
- URL: https://example-rolldice-mcp.local
- Functionality: Accepts JSON requests for roll/die commands, returns structured outcomes for agents.

## 2. Bootcamp AI Agent
- Purpose: Dedicated training and mentoring agent for workshop tasks, including code scaffolding and guidance.
- URL: https://example-bootcamp-agent.local
- Functionality: Supports agent orchestration, task execution, and stateful session management.

## 3. Calendar Booking
- Purpose: Scheduling MCP server for meeting/time-block booking from conversation context.
- URL: https://example-calendar-booking.local
- Functionality: CRUD for events, availability lookups, timezone-aware scheduling.

## 4. GitHub
- Purpose: Source control MCP server for repository read/write, PR management, commits, and CI interactions.
- URL: https://api.github.com
- Functionality: Uses GitHub MCP authentication tokens and handles repo file edits via API.
